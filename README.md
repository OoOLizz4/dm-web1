Explorons les villes de France
par Lisa Verrier.

Ce site permet de visualiser une carte et d'afficher des villes de France selon leur nom. 
On peut choisir si on veut que le nom contienne une expression, commence par une expression ou finisse par celle-ci. 
Il y a aussi des boutons prédéfinis qui nous donnent une liste de villes dont le nom correspond à certains critères.
